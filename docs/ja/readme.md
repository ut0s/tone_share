# 使い方

## 送信（音を再生）

- 送信したいテキストを入力します
- 「再生して送信」を押します（スピーカーから音が出ます）

## 受信（マイクで解析）

- 「受信開始」を押します（ブラウザのマイク許可が必要です）
- 受信した内容は「受信ログ」に表示されます

## 注意

- 周囲が騒がしいと誤検出しやすいです
- 端末/ブラウザによっては、音声開始にユーザー操作が必要です

- 参考: [GitHub - ggerganov/ggwave: Tiny data-over-sound library](https://github.com/ggerganov/ggwave)
